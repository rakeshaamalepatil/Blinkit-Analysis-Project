Blinkit Data Analysis

A project showcasing key insights from the Blinkit dataset.

Project Overview

This project is a guided data analysis of the Blinkit dataset. It involves cleaning and standardizing data, calculating key performance indicators (KPIs) like total sales and average ratings, and creating visualizations. A key insight from the project is that 'Low Fat' items account for the majority of total sales. The project demonstrates proficiency in using popular Python libraries such as pandas, numpy, and matplotlib for data analysis.

Key Findings

  • **Data Cleaning**: Standardized the `Item_Fat_Content` column to ensure consistency.
  • **KPIs**: Calculated key business metrics, including total sales, average sales, and average product ratings.
  • **Visualizations**: Visualizations were created to highlight key insights, such as total sales by fat content and location.

1. Standardizing Item Fat Content


# Standardize 'Item_Fat_Content' values
df['Item_Fat_Content'] = df['Item_Fat_Content'].replace(
    ['low fat', 'LF', 'reg'],
    ['Low Fat', 'Low Fat', 'Regular']
)
print(df['Item_Fat_Content'].value_counts())

2. Visualizing Sales by Outlet Location


# Group by location and sort by sales
sales_by_location = df.groupby('Outlet_Location_Type')['Sales'].sum().reset_index()
sales_by_location = sales_by_location.sort_values('Sales', ascending=False)

# Create a bar plot
plt.figure(figsize=(8, 3))
sns.barplot(x='Sales', y='Outlet_Location_Type', data=sales_by_location)
plt.title('Total Sales by Outlet Location Type')
plt.xlabel('Total Sales')
plt.ylabel('Outlet Location Type')
plt.show()
